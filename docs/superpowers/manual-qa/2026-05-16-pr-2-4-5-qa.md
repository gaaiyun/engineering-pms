## E. 失败处理

| 现象 | 应对 |
|---|---|
| APK 装上后启动闪退 | 检查 logcat，搜 `RealtimePlugin` 报错。回退 v2.97 直到诊断完。 |
| 持久通知不出现 | 通知权限未授予 / IMPORTANCE_LOW 频道被用户禁用 |
| 后台不收消息但持久通知还在 | 大概率国产 ROM 杀进程 → 加白名单 |
| 后台不收消息且持久通知消失 | Service 被杀 → 调高白名单等级 |
| dev server 桌面表格不显示 | 浏览器宽度 < 1024px ；缩放级别影响 CSS px |
| 批量删除卡住 | 单个 PB delete 出错 → 检查浏览器 console 看 mutation 报错 |
